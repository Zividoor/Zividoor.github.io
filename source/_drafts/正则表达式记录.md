注释转换（// 转换成 /**/）

```
// (.*)

/**\n\t\* $1\n\t\*\/
```

属性转json

```
/\*\*(.*)\n(.*) (.*)\n(.*)\n(.*)private (.*) (.*);

,{\nname : '$7', //$3\n \ttype : '$6'\n}
```

添加数据Comment

```
/\*\*(.*)\n(.*) (.*)\n(.*)\n(.*)@Column\(name = "(.*)"(.*)\n(.*)
COMMENT ON COLUMN CSC_CSTMGMT\.CST_APPORTION_INFO\.$6 IS '$3';

```
